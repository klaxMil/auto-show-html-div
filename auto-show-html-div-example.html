Buttons

<button type="button" class="btn">

Button Styles

Anything that is given a class of .btn will inherit the default look of a gray button with rounded corners.
We can change color adding the class:
  • .btn-primary.
    • primary action
  • .btn-success.
    • successful or positive action
  • .btn-info.
    • informational alert messages
  • .btn-warning.
    • caution should be taken with this action
  • .btn-danger.
    • dangerous or potentially negative action
  • .btn-link.
    • Deemphasize a button by making it look like a link while maintaining button behavior

Button Sizes

Change the size adding the class:
  • .btn-lg.
    • large
  • .btn-sm.
    • small
  • .btn-xs.
    • extra small
  • .btn-block.
    • block level: span the full width of a parent

Button State

  • .active.
    • Active: Buttons will appear pressed (with a darker background, darker border, and inset shadow)
  • disabled="disabled" .
    • When you disable a button, it will fade in color by 50%, and lose the gradient.

Button Groups

Bootstrap allows you to group a series of buttons together (on a single line) in a button group
  • Use a <div> element with class .btn-group to create a button group
  • Add class .btn-group-lg|sm|xs to size all buttons in the group
  • To span the entire width of the screen, use the .btn-group-justified class with <a> elements of class .btn:

Vertical Button Groups

  • Use the class .btn-group-vertical to create a vertical button group
  • Add class .btn-group-lg|sm|xs to size all buttons in the group

Drop down menu button

  • Use a .btn-group with a button class="btn ... dropdown-toggle" data-toggle="dropdown" and a list of links to create a drop-down menu button
  • Split the button into two buttons using span class="caret" as button for drop-down menu (put a &nbsp; after the </span> to preserve height)
  • Nest button groups to create groups with drop-down menus
  • Use glyphicons to decorate your buttons (or to make a graphic button)
